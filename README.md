# DG Журнал

Журнал domgazobeton.com — практические разборы о доме из газобетона.

Чистый HTML+CSS+JS без сборщика. Хостится на GitHub Pages, доступен по адресу `journal.domgazobeton.com`.

## Структура

- `index.html` — разводящая страница (главная)
- `<slug>.html` — статьи
- `assets/css/` — стили
- `assets/js/` — скрипты
- `assets/img/` — картинки
- `sitemap.xml`, `robots.txt`, `404.html`, `CNAME` — служебное

## Добавить новую статью

1. Создать файл `<slug>.html` по образцу одной из существующих статей
2. Положить картинки в `assets/img/`
3. Добавить URL в `sitemap.xml`
4. Закоммитить и запушить — деплой через 30 секунд
